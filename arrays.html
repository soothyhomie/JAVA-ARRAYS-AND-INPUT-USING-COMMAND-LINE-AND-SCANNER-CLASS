An array is a collection of similar type of elements which has a contiguous memory location.
Types of array in java are:

    1. Single Dimensional Array(1D array)
    2. Multidimensional Array(2D array)

1-D array

1D array or single dimensional array contains only a row.





Declaration/ memory allocation of 1D array

datatype arrayName[]=new datatype[size];
Or
datatype[] arrayName=new datatype[size];

Example:
int arr[]= new int[10];
int[] arr= new int[10];



Meaning of each word in above statement

datatype It is the type of element we want to enter in the array like int,float,double etc.
arrayName It is an identifier
new It is a keyword that creates an instance in the memory
size It is the length of the array



Alternatively in situation where the size of array and variable are already known we can do,

int arr[]={10,20,30} //declaration,instantiation,initialization

We can take input in an array by the two following ways:

    1. Using Command Line
    2. Using Scanner Class







2D array

A two-dimensional array is an array that contains elements in the form of rows and columns.
It means we need both row and column to populate a two-dimensional array.
Matrix is the best example of a 2D array.





We can declare a two-dimensional array by using the following statement:

datatype arrayName[][] = new datatype[m][n];
datatype[][] arrayName[][] = new datatype[m][n];

Example:
int arr[][]= new int[10][20];
int[][] arr= new int[10][20];


Meaning of each word in above statement

datatype It is the type of element we want to enter in the array like int,float,double etc.
arrayName It is an identifier
new It is a keyword that creates an instance in the memory
m It is the number of rows
n It is the number of column



Alternatively in situation where the size of array and variable are already known we can do,

int arr[2][2]={10,20,30,40} //declaration,instantiation,initialization with 2 row and 2 column

We can take input in an array by the two following ways:

    1. Using Command Line
    2. Using Scanner Class
